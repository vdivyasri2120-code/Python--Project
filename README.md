# Python--Project
                        🛒 Sales Performance Analysis Project
📌 Project Overview:-

This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python.
The objective is to analyze overall sales performance, profitability trends, regional distribution, and the impact of discounts using data visualization techniques.

The project helps in understanding business performance through structured analysis and graphical representation.

🎯 Objective:-

To analyze sales performance, profit trends, regional distribution, and relationships between financial variables in order to support better business decision-making.

🛠 Tools & Technologies Used:-

Python

Pandas – Data Cleaning & Analysis

Matplotlib – Data Visualization

Seaborn – Statistical Visualization

Jupyter Notebook

📂 Dataset Description-

The dataset contains transactional sales records with the following key features:

Order ID

Order Date

Customer Name

Region

City

Category

Sub-Category

Product Name

Quantity

Unit Price

Discount

Sales

Profit

Payment Mode

📊 Analysis Performed:-
1️⃣ Data Cleaning

Checked for missing values

Handled null values

Removed duplicates

Verified data types

2️⃣ Sales Analysis

Calculated Total Sales

Analyzed Sales by Category (Bar Chart)

Analyzed Sales by Region (Pie Chart)

Evaluated category-wise performance

3️⃣ Profit Analysis

Calculated Total Profit

Visualized Profit Distribution using:

Histogram

KDE Plot

Analyzed Profit by Category using Violin Plot

Identified profit variation across different product groups

4️⃣ Time-Based Analysis

Analyzed Sales Over Time (Line Graph)

Analyzed Profit Trends Over Time

Observed fluctuations and performance patterns

5️⃣ Correlation Analysis

Created a Heatmap to analyze relationships between:

Sales

Profit

Discount

Quantity

Unit Price

This helped identify positive and negative correlations between financial variables.

📈 Key Insights

Sales performance varies significantly across categories.

Certain regions contribute more to overall revenue.

Profit distribution shows variability, including lower-margin transactions.

Discount has an impact on profitability.

Sales and Profit show a strong positive relationship.

Quantity and Unit Price influence overall revenue.

Time-based trends indicate fluctuating sales performance.

✅ Conclusion:-

This project demonstrates how data analysis and visualization can uncover valuable business insights.

The analysis shows that:

Revenue is concentrated in specific product categories.

Regional performance differs, indicating growth opportunities.

Profitability is influenced by pricing and discount strategies.

Strong correlation exists between sales and profit.

Monitoring sales trends over time is essential for forecasting and inventory planning.

Overall, this project highlights the importance of Exploratory Data Analysis (EDA) in understanding business performance and supporting strategic decision-making.
